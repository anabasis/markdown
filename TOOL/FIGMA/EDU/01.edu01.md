## Plugin

- Plugin > Manage Plugins > Plugins : 플러그인 설치
- Unsplash : 저작권없는 무료 이미지 사이트 추천

## 이미지 오버레이

- Design > Fill > + > Linear : 그레디언트
- Design > Fill > + > Solid : Solid

### [Rectangle]

- Design > Constrains > X:Center, Y:Center
- Design > Stroke > + 

### [Text]
- Shift + Alt + 마우스 : 오브젝트 복사
- Alt + 마우스 : 속성값 변경
